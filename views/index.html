IMGGET Service

Usage:

To search for images: GET https://imgget.glitch.me/?searchTerm=<YOUR_TERM>&offset=<OFFSET>
Results are indexed from 1 to n, with n being the total number of results.
Results are shown in batch of 10. That is, if offet=1, server will response with result number 1 to 10.
To view the latest 10 searches: GET https://imgget.glitch.me/history

Note: One can only make 100 request per day.
